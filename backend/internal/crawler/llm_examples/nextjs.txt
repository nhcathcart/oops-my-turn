# Next.js

> Next.js is a React framework for building full-stack web applications. It combines React components with routing, rendering, caching, deployment, and optimization features for production-ready web apps.

The official Next.js llms.txt works best as a gateway document rather than a giant page-by-page catalog. It points models toward the main documentation index, version-specific docs, support policy, learning materials, and recent blog updates.
